# 📱 Flipkart Mobile Web Scraper

A Python-based web scraping project that extracts mobile phone data from Flipkart, specifically targeting mobiles in the ₹20,000 to ₹30,000 price range. This project is useful for data analysis, price tracking, and building datasets for machine learning or research purposes.

##🚀 Features
Scrapes mobile listings from Flipkart
Filters mobiles within ₹20,000 – ₹30,000 price range
## Extracts:
- 📌 Product Name
- 💰 Price
- ⭐ Ratings
- 📝 Specifications
  - Handles pagination
  - Stores data in CSV / JSON format
  - Clean and structured output
## 🛠️ Tech Stack
- Python 🐍
- BeautifulSoup
- Requests
Pandas
